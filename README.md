\# RC-Helicopter-107g-Leap



Control an RC Helicopter Toy Using the Leap Motion Controller.



Experience the future of interactive play by controlling an RC helicopter with simple hand movements! This innovative setup uses the Leap Motion Controller, a gesture-sensing device, to detect the position and motion of your hands in real time. By moving your hands in the air, you can lift, lower, rotate, and maneuver the helicopter with precise control, eliminating the need for traditional joysticks or remote controls.



The system translates natural gestures into flight commands, making it intuitive for users of all ages, from beginners to tech enthusiasts. With this combination of gesture recognition technology and a classic RC helicopter toy, you get a hands-on, futuristic flying experience that blends fun, learning, and innovation. Ideal for experimenting with robotics, motion sensing, and interactive electronics.

\## 🚀 Installation and Setup



1\. Setup Breadboard with Arduino following the circuit pattern.

&nbsp;  - Connect 1 LED to pin 13 \& GND - for STATUS LED

   - Connect 1 jumper wire to pin 8 - for LED and IR LED

   - Connect 2 jumper wires to pin V5 \& GND - for power

&nbsp;        

2\. Install and setup Leap Motion Controller software.



3\. Connect Leap Motion Controller and Arduino via USB to your computer.

4\. Test Leap Motion Controller\*\* to ensure it is working properly.

5\. Open and run Arduino program:

   - Verify \& Compile code

   - \*\*Note:\*\* Ensure no jumper wires are connected to Arduino before uploading. Connect jumper wires only after Verify \& Compile.

6\. Install Processing software ( download here https://github.com/JeloydD/processing-3.2.1 ).

7\. Open project and run Processing program.

8\. Position Leap Motion Controller: green light facing you.

9\. Place your hand above the Leap Motion sensor. The system should respond to your gestures.



Note: To check if working, LED should blinking light faster and not on steady mode.
